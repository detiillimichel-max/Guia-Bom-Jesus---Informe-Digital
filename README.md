Como publicar no GitHub PELO CELULAR (sem codar)
No app do GitHub, clique em + > New repository
Nome: guia-bom-jesus
Marque Public
Create
Clique em Add file > Upload files
Selecione o arquivo index.html que você baixou aqui
Commit changes
Vá em Settings > Pages
Source: Deploy from a branch
Branch: main / root
Save
Em 1 minuto seu site estará no ar em:
https://SEU-USUARIO.github.io/guia-bom-jesus/

Como editar um telefone depois
No GitHub, abra o index.html > clique no lápis > procure o nome > troque o número > Commit. Pronto, site atualiza.
